# Galway Geometry

Geometry made visible.

Galway Geometry is an all-HTML visual geometry project centred on lawful construction, geometric aesthetics, and the emergence of a field from simple relations.

The project begins from a strict idea:

A relation is not introduced. A relation is produced.

The repository is designed as a public-facing HTML project. It can hold visual theorem readers, exhibitions, geometry canvases, drawing tools, colouring tools, mandala studies, and learning pages without requiring a build system.

## What this is

Galway Geometry is a place to:

- watch geometric relations compose and decompose;
- explore equilateral, hexagonal, circular, and triangular field structures;
- draw and colour on lawful geometric substrates;
- create geometric mandalas and tessellations;
- study how a field emerges from repeated construction;
- collect visual plates as an exhibition catalogue.

## Public layers

Watch: open a visual geometry reader and let relations unfold.

Draw: use the field as a canvas for colouring, mandalas, and geometric studies.

Build: create new plates using the same substrate grammar.

Explore: study how points, circles, cells, hexagons, and fields emerge from lawful repetition.

## Design rules

The project should stay clean, crisp, readable, and responsive on phone, tablet, and desktop.

The interface should not compete with the geometry.

Every visual tool should prefer directness over clutter.

Every construction should preserve the distinction between decoration and geometry.

## Current status

This repository is a public backbone. The next phase is to add HTML modules for exhibitions, field canvas experiments, visual readers, and workshops.
